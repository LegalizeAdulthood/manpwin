Parameter Files and Command Line Options

Parameter files are used to save favourite areas and parameters for fractals currently being processed. Their main advantage is that they take very little room to store.

The main parameters are:

-cScreen dimensions Xleft, Ytop, Height

-mScreen processing method:

1=1pass

2=2pass

G=Guessing followed by passes

B=Boundary Tracing

T=Terresal

H=Horizontal Blinds

V=Vertical Blinds

-jIf present indicates Julia set. x,y location follows flag

-tThreshold beyond which iteration halts 1 - 32000

-iThe 'inside' colour: 0 - 255

-3Plot as 3D. This is followed by:

Scaling x

Scaling y

Scaling z

Rotation x (deg)

Rotation y (deg)

Rotation z (deg)

-uTrue Colour Palette. This is followed by:

Start Red Colour

Start Green Colour

Start Blue Colour

Red Increment value

Green Increment value

Blue Increment value

-sSaveAs filename (in PNG foprmat)

-oStereo pairs

-eExit after completion

-pPeriodicity

-rAutoStereo depth

-bBiomorph colour

-kNumber of decomposition colours (power of 2)

-lLogarithmic palette (followed by starting value)

-aPalette filename
